# 输入信息收集清单（image2pencil）

在开始绘制前，先检查以下信息是否具备。

## 1) 必填项（缺一不可）

- 参考输入：至少 1 张截图或设计稿
- 任务目标：复刻新页面 / 修改已有页面
- 目标终态：结构级、高保真、像素级（三选一）

## 2) 强烈建议项（影响质量）

- 平台：Web / Mobile
- 目标尺寸：如 1440x900、375x812
- 字体偏好：系统默认或指定字体
- 组件状态：默认 / hover / active / disabled
- 必须一致区域：例如表单区、按钮区、标题区

## 3) 设计文档需求

- 文档放置位置：默认在页面右侧
- 文档粒度：简版 / 详细版
- 是否包含：
  - 字段说明
  - 校验规则
  - 交互流程
  - 视觉规范
  - 待确认项

## 4) 快速补问模板

当信息不足时可直接发：

1. 你希望复刻到什么精度（结构级 / 高保真 / 像素级）？
2. 页面目标尺寸是多少（Web 或 Mobile）？
3. 哪些区域必须与截图完全一致？
4. 是否需要包含 hover/选中/禁用等状态？
5. 设计文档要简版还是详细版？
