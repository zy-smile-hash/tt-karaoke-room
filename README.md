# TT语音K歌房 - 交互演示

这是一个基于HTML/CSS/JavaScript实现的TT语音K歌房移动端交互演示页面。

## 功能特性

- 18个功能页面的跳转演示
- 流畅的页面切换动画
- 返回按钮导航
- **交互式热点编辑器**（可拖动调整按钮位置）
- 本地存储保存位置

## 使用方法

### 基础演示模式
1. 直接用浏览器打开 `index.html` 即可查看
2. 点击首页上的不同功能区域进行跳转
3. 点击左上角返回按钮回到首页

### 热点位置调整（重点！）
由于截图可能包含空白区域，需要调整热点位置以匹配实际按钮：

#### 步骤1：进入编辑模式
1. 在右侧找到「🎯 热点编辑器」面板
2. 点击「✏️ 进入编辑模式」按钮
3. 页面底部会出现"编辑模式"提示
4. 所有热点区域会显示蓝色边框

#### 步骤2：选择并调整热点
1. 点击任意热点区域（或编辑器中的列表项）选中它
2. 选中的热点会显示蓝色高亮边框
3. 直接在设计图上拖动热点到正确位置
4. 或者在右侧面板手动输入 Top、Left、Width、Height 值

#### 步骤3：保存位置
1. 调整完成后，点击「💾 保存坐标」按钮
2. 位置会自动保存到浏览器本地存储
3. 刷新页面后位置依然有效

#### 步骤4：退出编辑模式
1. 点击「✅ 退出编辑模式」按钮
2. 热点边框消失，进入演示模式

### 导出CSS代码
如果需要永久保存位置，可以：
1. 点击「📋 复制CSS」按钮
2. CSS代码会自动复制到剪贴板
3. 将代码粘贴到HTML文件的 `<style>` 部分替换原有的 `.hotspot-xxx` 样式

### 重置位置
如果需要恢复默认位置，点击「🔄 重置」按钮即可。

## 技术说明

- 使用纯原生 HTML/CSS/JavaScript，无需依赖
- 响应式设计，适配移动端显示（375×812px iPhone尺寸）
- 热点区域覆盖法实现点击交互
- opacity 过渡实现页面切换动画
- localStorage 保存用户调整的位置
- 支持鼠标和触摸操作

## 浏览器兼容性

- Chrome/Edge (推荐)
- Firefox
- Safari
- 移动端浏览器

## 注意事项

1. 首次使用建议先进入编辑模式确认热点位置
2. 位置调整后记得保存，否则刷新会丢失
3. 如果分享给他人，建议导出CSS代码并更新到HTML文件中
4. localStorage 存储空间有限，调整后的位置仅在本地有效
